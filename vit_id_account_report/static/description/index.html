Template Generic Laporan Keuangan Indonesia

Module ini gunanya untuk membuat template Financial Report (Neraca dan Laba Rugi) untuk perusahaan di Indonesia. Berdasarkan template ini, anda bisa kembangkan lebih lanjut untuk disesuaikan dengan format laporan keuangan masing-masing perusahaan.

Addons ini bisa anda peroleh secara gratis jika mengikuti Training Odoo di vitraining.com atau membeli E-BOOK "PANDUAN LENGKAP IMPLEMENTASI ODOO V10 UNTUK PERUSAHAAN RETAIL DI INDONESIA" di shop.vitraining.com maupun di Google Play.

Neraca

Laporan Neraca format Indonesia minimal harus menampilkan: Aktiva, Hutang dan Modal, Laba Rugi Tahun Berjalan yang diambil dari laporan Laba Rugi

Terdapat pilihan financial report Neraca ketika akan mencetak laporan keuangan

Hirarki laporan Neraca dapat anda custom lebih lanjut sesuai kebutuhan masing-masing perusahaan

Laporan Neraca dalam print out PDF

Laba Rugi

Laporan Laba Rugi format Indonesia minimal harus menampilkan: Pendapatan, HPP, Laba Rugi Kotor, Biaya Admin dan Umum, Pendapatan Lain, Biaya Lain, Laba Rugi Bersih yang perlu dimunculkan di Neraca

Terdapat pilihan financial report Lab Rugi ketika akan mencetak laporan keuangan

Hirarki laporan Lab Rugi dapat anda custom lebih lanjut sesuai kebutuhan masing-masing perusahaan

Laporan Laba Rugi dalam print out PDF
